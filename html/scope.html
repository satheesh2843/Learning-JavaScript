scopes in java script:

var :

in this keyword we can use same declared variable again and again without any error

let :

in this keyword we can only once declare the variable and we can change the value of variable

const :

in this keyword we can declare the variable only once and assign value once, we can't change value to variable

global scope :

it can used in entire program .

local scope

it can used in curly brackets only .

function scope

it can be used in function only.
